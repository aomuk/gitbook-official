插件简介

插件

GitBook插件遵循NPM包的定义。

gitbook-plugin是一个命令行工具，可以帮助您创建，测试和发布插件。

创建你的第一个插件

从NPM安装gitbook-plugin：

$ npm install gitbook-plugin -g

然后创建你的插件：

$ gitbook-plugin create

您将被要求一个插件名称，以及一些其他的内容来完成创建过程。
